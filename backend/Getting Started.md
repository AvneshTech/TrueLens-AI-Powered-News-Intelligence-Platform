# 🚀 Getting Started

Follow these steps to set up and run TrueLens locally.

## Prerequisites

Make sure you have installed:

- Java 21+
- Maven 3.9+
- Node.js 20+
- Python 3.10+
- MongoDB Atlas Account Or MongoDB Compass Installed
- Git
